abstract Parser<T>(Vector<Function<T>>)

no package

import Parsihax

The Parser object is a wrapper for a parser function. Externally, you use one to parse a string by calling var result = SomeParser.apply('Me Me Me! Parse Me!');

Variables

apply:Function<T>

Getting Parser.apply from a parser (or explicitly casting it to Function returns parsing function String -> ?Int -> Result<A> (or just Function), that parses the string and returns Result<A>.

Changing Parser.apply value changes parser behaviour, but still keeps it's reference, what is really usefull in recursive parsers.
